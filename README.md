# 회고(참고 링크 및 코드 개선)
    - 중복값을 제거하기 위한 목적은 아주 명확하지만, 개인적으로는 숫자 5개를 한꺼번에 입력하지 않고 하나씩 입력한 것은 다소 비효율적일 수도 있겠다는 생각을 했습니다.

    - 저희는 이 문제를 해결하기 위해 프린트문으로 "0부터 9까지 서로 다른 숫자를 5개 입력하세요!"라고 안내하고, list(map(int, input().strip())) 으로 list 형식으로 숫자를 받아 저장하였습니다. 다만, 저희는 5개 digit이 아닌 입력에 대한 부분은 고민해보지 않아서, 이 부분은 저희도 반영해야할 것 같다고 생각됩니다.
